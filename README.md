Module: Intro to Programming

Project:
This dashboard visualises the following key macroeconomic indicators for selected countries:
-Unemployment Rate
-GDP growth
-Interest Rate
-Inflation Rate

Each panel is interactive, allowing users to:
- Select a country  
- Choose a year range  
- Compare trends across indicators

Tools and Libraries: 
Pyhton
Streamlit - for building the dashboard
Pandas - for handling data
Plotly - for visualing charts
NumPy - for numerical analysis
